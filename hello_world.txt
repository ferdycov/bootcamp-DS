- First step to learn about .git

| Perintah Git               | Fungsi                                             |
| -------------------------- | -------------------------------------------------- |
| `git init`                 | Membuat repository Git baru di folder lokal        |
| `git clone <url>`          | Mengunduh repository dari server ke komputer lokal |
| `git status`               | Melihat status perubahan file dalam repository     |
| `git add <file>`           | Menambahkan file tertentu ke staging area          |
| `git add .`                | Menambahkan semua file ke staging area             |
| `git commit -m "pesan"`    | Menyimpan perubahan dari staging ke repository     |
| `git log`                  | Melihat riwayat commit                             |
| `git diff`                 | Melihat perbedaan perubahan pada file              |
| `git push origin <branch>` | Mengirim commit ke remote repository               |
| `git pull origin <branch>` | Mengambil dan menggabungkan perubahan dari remote  |
| `git fetch`                | Mengambil update dari remote tanpa merge           |
| `git branch`               | Melihat daftar branch                              |
| `git branch <nama_branch>` | Membuat branch baru                                |
| `git checkout <branch>`    | Berpindah ke branch lain                           |
| `git checkout -b <branch>` | Membuat dan langsung pin                           |

Workflow Git yang paling umum:
edit file → git add → git commit → git push
